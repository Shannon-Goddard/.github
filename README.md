# .github

This is the central configuration repository for **Shannon-Goddard** (Loyal9 LLC).

It holds default community health files, funding info, issue templates, and other settings that automatically apply to **all my repositories** (GrowApp, CI-Strains-Pro, Mission Mischief, etc.).

## What's Inside

- **FUNDING.yml** — GitHub Sponsors and other ways to support the work  
- **CONTRIBUTING.md** — How to contribute productively  
- **CODE_OF_CONDUCT.md** — Keeping the chaos respectful  
- **SUPPORT.md** — Where to get help  
- Issue templates (bug reports, feature requests, etc.)
